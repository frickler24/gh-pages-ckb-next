USB communication documentation for ckb-next in wireshark format as an example for the K95RGB.

Tipps to use:

  • Download the zip file and unzip it into a directory of your choice.
  • Open the files with wireshark (if necessary, you must install the application first).
  • Comments are inserted in the logs at relevant points. You can see this by the green lines (see picture).
  • Fold up the comment lines, some comments are longer than one line. Expanding just one layer should suffice.
