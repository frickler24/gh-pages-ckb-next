# gh-pages-ckb-next

![sea view](https://cloud.githubusercontent.com/assets/6292952/25564735/e48e7f92-2db9-11e7-8b51-9f83a4195d61.png)

First directory level of generated github-pages for ckb-next from mattanger

Use the script documentation/setupBranch-gh-pages.sh in repo ckb-next
to in stall these files in a correct manner into a fresh branch gh-pages
in ckb-next repo.

After installing a new branch "gh-pages", the content (html-content, man-tar-file and pdf-files)
are generated by doxygen via travis-ci.
